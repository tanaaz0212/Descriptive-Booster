# Descriptive-Booster
#  Income Distribution & Household Demographics Analysis

##  Project Overview

This project analyzes household demographic and income data using statistical techniques and visualizations. The goal is to understand income patterns, relationships between variables, and key statistical properties of the dataset.

---

##  Objectives

* Perform statistical analysis on household data
* Understand income distribution
* Apply measures of central tendency and dispersion
* Visualize data using charts
* Interpret real-world insights

---

##  Dataset Information

The dataset contains **household-level data** with the following columns:

| Column Name           | Description                      |
| --------------------- | -------------------------------- |
| Household_ID          | Unique ID for each household     |
| Age_of_Household_Head | Age of the head of the household |
| Household_Income      | Monthly household income         |
| Education_Level       | Education category               |
| Family_Size           | Number of family members         |
| Owns_House            | Yes/No                           |
| Urban_Rural           | Urban or Rural                   |

---

##  Tools Used

* Python (Pandas, Matplotlib, Seaborn)
* Jupyter Notebook
* Microsoft Excel

---

##  Analysis Performed

###  1. Data Understanding

* Classified variables into numerical and categorical

---

###  2. Central Tendency

* Mean
* Median
* Mode

---

###  3. Dispersion Measures

* Range
* Variance
* Standard Deviation
* Interquartile Range (IQR)

---

###  4. Distribution Analysis

* Histogram (Income Distribution)
* Skewness
* Kurtosis

---

###  5. Relationship Analysis

* Age vs Income (Scatter Plot)
* Income vs Education Level (Boxplot)

---

##  Visualizations

*  Histogram for income distribution
*  Boxplot for income vs education
*  Scatter plot for age vs income

---

##  Key Insights

* Income distribution is slightly right-skewed
* Higher education is associated with higher income
* Urban households generally earn more
* Some outliers exist in income data
